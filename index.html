Yeah, almost forgot.. list of git commands:

Setting ur name & email

git config --global user.name Your Name

git config --global user.email your_email@whatever.com

Creating repository and adding pages to it

git init - to initialize empty Git repository.

git add "file name.extension"(etc. index.html) - to start tracking file(also "-a" to skip adding modified files)

git commit -m(message which u will left as a reminder or note for future needs) - makes a save on current stage of project

Cheking status

git status - shows info, i hope it`s a right statement)

Project log

git log - shows history of all commits

Pushing our Project

git push -u shared/origin "branch name" - to send changes to remote repository

Типы селекторов

  1. По названию элемента(тэга) - p{color:blue;}
  2. По id - #selc_id{color: yellow;}
  3. По классу - .klass{color:aqua;}
  4. По нескольким классам - paint_it_red & span.kekw
  5. По вложенности - ol li p{color:violet}
  6. Универсальный селектор - *{color:brown}
  7. Соседние селекторы - q+cite{color:steelblue}
  8. Дочерние селекторы - li>a{color:royalblue}
  9. [атрибут="значение"] - a[target="wow"]{color:salmon;}
  10. [атрибут^="значение"] - a[href="http"]{color:tan}
